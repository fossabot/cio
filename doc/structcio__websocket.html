A pointer to a function which is called if a ping frame was received.

Library users are note required to set this function pointer. If you set this function pointer, please do NOT sent a pong frame as a response. The library takes care of this already.

Parameters
wsThe websocket which received the ping frame.
dataThe data the ping frame carried.
lengthThe length of data the ping frame carried.

void(* cio_websocket::onclose)(const struct cio_websocket *ws, enum cio_websocket_status_code status, char *reason, size_t reason_length)

A pointer to a function which is called when a close frame was received.

Library users are note required to set this function pointer. If you set this function pointer, please do NOT close() the websocket in the callback function. This is done immediately by the library after return of this function.

Parameters
wsThe websocket which received the pong frame.
dataThe data the pong frame carried.
lengthThe length of data the pong frame carried.
void(* cio_websocket::on_error)(const struct cio_websocket *ws, enum cio_websocket_status_code status)

A pointer to a function which is called if a receive error occurred.

Library users are note required to set this function pointer. If you set this function pointer, please operate no longer on this websocket (Do not call close() etc. on the websocket). Immediately after this function returns, the library closes the websocket on its own.

Parameters
wsThe websocket which encountered the error.
statusThe status code describing the error.
