typedef void(* cio_io_stream_read_handler_t) (struct cio_io_stream *io_stream, void *handler_context, enum cio_error err, struct cio_read_buffer *buffer)

The type of a function passed to all cio_io_stream read callback functions.

Parameters
io_streamThe cio_io_stream the read operation was called on.
handler_contextThe context the functions works on.
errIf err != CIO_SUCCESS, the read operation failed, if err == CIO_EOF the peer closed the stream.
bufferThe buffer that was filled.

The type of a function passed to all cio_io_stream write callback functions.

Parameters

bytes_transferredThe number of bytes transferred.
