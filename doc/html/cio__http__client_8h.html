typedef void(* cio_response_written_cb) (struct cio_http_client *client, enum cio_error err)

The type of a response callback function passed to the write_response function.

Parameters
clientThe cio_http_client that wrote the response.
errAn error code. If err != CIO_SUCCESS, the client will be closed automatically. So please don't close the client in the callback function. Any other resources not associated with client should be cleaned up here.
