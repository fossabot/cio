Detailed Description

This file contains the interface of a timer.

Currently only one-shot timers are supported, no periodic timer. If you need a periodic timer, you have the rearm the timer in your timer callback.

◆ cio_timer_close()

void cio_timer_close ( struct cio_timer *  timer)

Closes a timer and frees underlying resources.

If the timer is armed and has not expired yet, the timer will be canceled and the timer callback will be called. If a close_hook was given in cio_timer_init, the hook is called.

Parameters
timerA pointer to a struct cio_timer which shall be closed.

Initializes a cio_timer.

Parameters
timerThe cio_timer that should be initialized.
loopThe event loop the timer shall operate on.
close_hookA close hook function. If this parameter is non NULL, the function will be called directly after closing the cio_timer. It is guaranteed that the cio library will not access any memory of cio_timer that is passed to the close hook. Therefore, the hook could be used to free the memory of the timer struct.
Returns
CIO_SUCCESS for success.
