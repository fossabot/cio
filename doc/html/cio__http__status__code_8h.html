enum cio_http_status_code

The cio_http_status_code enum lists all HTTP status codes that can be emmited by the cio_http_server.

Enumerator
CIO_HTTP_STATUS_SWITCHING_PROTOCOLS 

The requester has asked the server to switch protocols and the server has agreed to do so.

CIO_HTTP_STATUS_OK 

Standard response for a successful HTTP request.

CIO_HTTP_STATUS_BAD_REQUEST 

Request not processed due to a client error.

CIO_HTTP_STATUS_NOT_FOUND 

The requested resource was not found.

CIO_HTTP_STATUS_TIMEOUT 

The request was not completed in a certain time.

CIO_HTTP_STATUS_INTERNAL_SERVER_ERROR 

An internal server error occured.
