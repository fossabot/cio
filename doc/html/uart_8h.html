enum cio_error cio_uart_close ( struct cio_uart *  port)

Closes a UART port.

Once a UART has been closed, no further communication is possible. Closing the UART also closes the UART's cio_io_stream.

Parameters
portThe port to be closed.
Returns
CIO_SUCCESS on success.

◆ cio_uart_get_ports()

enum cio_error cio_uart_get_ports ( struct cio_uart  ports[],
size_t  num_ports_entries,
size_t *  num_detected_ports 
)

Detects the UART ports.

Detects the UART ports in a system and fills them into an array of cio_uart ports. If there are more ports in a system then provided by num_port_entries, only upto num_port_entries will be filled into ports.

Parameters
[out]portsA buffer array filled with the detected UART ports.
num_ports_entriesThe size of the ports buffer.
[out]num_detected_portsThe number of UARTs filled into ports.
Returns
CIO_SUCCESS on success.

◆ cio_uart_init()

enum cio_error cio_uart_init ( struct cio_uart *  port,
struct cio_eventloop *  loop,
cio_uart_close_hook_t  close_hook 
)

Initializes a UART port.

Parameters
portThe port to be initialized.
loopThe event loop the socket shall operate on.
close_hookA close hook function. If this parameter is non NULL, the function will be called directly after closing the cio_uart. It is guaranteed the the cio library will not access any memory of cio_uart that is passed to the close hook. Therefore the hook could be used to free the memory of the UART.
Returns
CIO_SUCCESS on success.
