enum cio_error(* cio_socket::close)(struct cio_socket *socket)

Closes the cio_socket.

Once a socket has been closed, no further communication is possible. Closing the socket also closes the socket's cio_io_stream.

Parameters
socketA pointer to a cio_socket which shall be closed.

enum cio_error(* cio_socket::set_keep_alive)(struct cio_socket *socket, bool on, unsigned int keep_idle_s, unsigned int keep_intvl_s, unsigned int keep_cnt)

Enables/disables TCP keepalive messages.

Parameters
socketA pointer to a cio_socket for which TCP keepalive should be changed.
onWhether or not to enable TCP keepalives.
keep_idle_sTime in seconds the connections needs to remain idle before start sending keepalive messages. This option might be unused in some platform implementations.
keep_intvl_sTime in seconds between individual keepalive probes. This option might be unused in some platform implementations.
keep_cntThe maximum number of keepalive probes before dropping the connection. This option might be unused in some platform implementations.
Returns
CIO_SUCCESS for success.
