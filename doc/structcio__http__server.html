enum cio_error(* cio_http_server::register_location) (struct cio_http_server *server, struct cio_http_location *location)

Register a location to the HTTP server.

If you register overlapping locations like "/foo" and "/foo/bar", the HTTP server looks for the best match with respect to the path in the HTTP request.

Parameters
serverThe HTTP server instance which shall serve the new location.
locationThe location that should be served by the HTTP server.
Returns
CIO_SUCCESS if the location was registered correctly.
