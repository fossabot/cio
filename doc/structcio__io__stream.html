enum cio_error(* cio_io_stream::close)(struct cio_io_stream *io_stream)

Closes the stream.

Implementations implementing this interface are strongly encouraged to flush any write buffers and to free other resources associated with this stream.

Parameters
io_streamA pointer to the cio_io_stream of the on which the operation should be performed.
Returns
CIO_SUCCESS for success.

enum cio_error(* cio_io_stream::write_some)(struct cio_io_stream *io_stream, const struct cio_write_buffer *buf, cio_io_stream_write_handler handler, void *handler_context)

Writes upto count buffers to the stream.

handler might be called if only parts of buffer had been written.

Parameters
io_streamA pointer to the cio_io_stream of the on which the operation should be performed.
bufThe buffer where the data is written from. Please note that the memory buf points to must be retained until handler was called.
countThe number of to write.
handlerThe callback function to be called when the write request is (partly) fulfilled.
handler_contextA pointer to a context which might be useful inside handler.
Returns
CIO_SUCCESS for success.
