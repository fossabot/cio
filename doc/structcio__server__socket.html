Parameters

enum cio_error(* cio_server_socket::bind)(struct cio_server_socket *ss, const char *bind_address, uint16_t port)

Binds the cio_server_socket to a specific address.

Parameters
ssA pointer to a cio_server_socket on which the bind should be performed.
bind_addressThe IP address a cio_server_socket shall bound to. If NULL, then cio_server_socket binds to all interfaces.
portThe TCP port the cio_server_socket shall listen on.
