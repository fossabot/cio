Detailed Description

This file contains the interface of a socket.

You can get an I/O stream from a socket, close the socket or set several socket options.

Initializes a cio_socket.

Parameters
sThe cio_socket that should be initialized.
loopThe event loop the socket shall operate on.
close_hookA close hook function. If this parameter is non NULL, the function will be called directly after closing the cio_socket. It is guaranteed the the cio library will not access any memory of cio_server_socket that is passed to the close hook. Therefore the hook could be used to free the memory of the server socket.
Returns
CIO_SUCCESS for success.
