enum cio_error(* cio_timer::expires_from_now)(struct cio_timer *timer, uint64_t timeout_ns, timer_handler handler, void *handler_context)

Set the timer's expiration time relative to now and arms the timer.

Parameters
timerA pointer to a struct cio_timer which shall expire.
timeout_nsThe expiration time relative to "now" in nanoseconds.
handlerThe callback function to be called when the timer expires or was cancelled.
handler_contextA pointer to a context which might be useful inside handler.
Returns
CIO_SUCCESS if timer was armed successfully.
enum cio_error(* cio_timer::cancel)(struct cio_timer *timer)

Cancels an armed timer.

Parameters
timerA pointer to a struct cio_timer which shall be canceled.
Returns
CIO_SUCCESS for success, CIO_NO_SUCH_FILE_OR_DIRECTORY if the timer wasn't armed.
void(* cio_timer::close)(struct cio_timer *timer)

Closes a timer and frees underlying resources.

If the timer is armed and has not expired yet, the timer will be canceled and the timer callback will be called. If a close_hook was given in cio_timer_init, the hook is called.

Parameters
timerA pointer to a struct cio_timer which shall be closed.
